# Holixora Lead Tracker Format

## Purpose
A simple sales tracking structure for Holixora to manage outreach, follow-ups, conversations, proposals, and pipeline movement.

## Recommended Tool Options
### Simple start
- Google Sheets
- Airtable
- Notion database

### Recommendation
Start with **Google Sheets** or **Airtable**.
Keep it simple early on.

---

## Core Pipeline Stages
Use these stages consistently:

1. Target Identified
2. First Outreach Sent
3. Follow-Up Sent
4. Conversation Started
5. Discovery / Qualification
6. Proposal / Recommended Path
7. Won
8. Lost
9. Nurture / Later

---

## Main Tracker Columns
### Basic lead identity
- Lead ID
- Company Name
- Contact Name
- Role / Title
- Industry Segment
- Website
- Location

### Qualification context
- Target Segment (Hotel / Restaurant / Café / Property / Related Service)
- Lead Source (Warm / Referral / Direct Outreach / Website / WhatsApp / Telegram / LinkedIn)
- Priority Level (High / Medium / Low)
- Fit Score (1–5)

### Sales activity
- Current Stage
- First Contact Date
- Last Contact Date
- Next Follow-Up Date
- Outreach Channel
- Outreach Message Used

### Need / pain context
- Main Pain Point
- Current Workflow Problem
- Opportunity Type (Audit / Automation / Software / Hospitality Solution / MVP)
- Potential Value Level (Low / Medium / High)

### Commercial tracking
- Proposed Next Step
- Proposal Sent? (Yes / No)
- Estimated Deal Size
- Decision Timeline
- Probability %

### Notes
- Last Conversation Summary
- Objections / Concerns
- Next Action
- Owner

## Suggested Spreadsheet Layout
### Tab 1 — Active Leads
Track all live leads currently in progress.

### Tab 2 — Won Deals
Track successful clients and completed conversions.

### Tab 3 — Lost / Paused
Track leads that did not move forward or should be revisited later.

### Tab 4 — Outreach Log
Track what messages were sent, through which channel, and when.

---

## Stage Definitions
### 1. Target Identified
The lead has been identified but no outreach has been sent yet.

### 2. First Outreach Sent
Initial message has been sent.

### 3. Follow-Up Sent
At least one follow-up has been sent after the first outreach.

### 4. Conversation Started
The lead has responded or a real conversation has begun.

### 5. Discovery / Qualification
You are learning about workflows, pain points, and fit.

### 6. Proposal / Recommended Path
You have suggested an audit, automation project, software build, or other next step.

### 7. Won
The lead has become a client or active project.

### 8. Lost
The opportunity is closed and unlikely to move forward.

### 9. Nurture / Later
The lead may still be valuable later, but timing is not right now.

---

## Fit Scoring Guide
### 5 — Strong fit
Clear hospitality relevance, visible operational pain, decision-maker access, good project potential.

### 4 — Good fit
Strong business relevance and likely project potential, but still needs validation.

### 3 — Moderate fit
Possible opportunity, but unclear urgency or unclear pain.

### 2 — Weak fit
Low urgency, weak alignment, or unclear budget/need.

### 1 — Poor fit
Not aligned with Holixora focus or unlikely to convert.

## Operating Rules
### Rule 1
Every lead must always have a **current stage**.

### Rule 2
Every active lead should have a **next action**.

### Rule 3
Every active lead should have a **next follow-up date**.

### Rule 4
Update notes after every meaningful conversation.

### Rule 5
If a lead is not progressing, move it to **Nurture / Later** instead of letting it become invisible.

---

## Recommended Next Workflow
Once this tracker exists, the sales workflow becomes:

1. identify lead
2. send outreach
3. track response
4. qualify pain and fit
5. propose next step
6. follow up consistently
7. convert or nurture
