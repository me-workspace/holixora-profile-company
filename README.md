# Holixora Company Profile Website

A Next.js company profile website for **Holixora Digital Technology**.

## Brand Direction
- Industry focus: hospitality and related service businesses
- Core offering: AI automation and custom software solutions
- Tagline: **Smart Systems for Modern Hospitality**
- Style: modern tech / premium

## Pages
- Home
- About Us
- Services
- Industries
- Contact

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm start
```
